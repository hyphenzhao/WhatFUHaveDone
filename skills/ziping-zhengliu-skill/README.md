# 子平正解·书籍蒸馏技能

> 基于《渊海子平》蒸馏笔记的八字命理分析技能

## 概述

本技能将《渊海子平》的知识通过"仓颉五维度蒸馏法"（框架、原则、案例、反例、术语）提取为可执行规则，提供完整的八字排盘、格局判断、大运流年分析功能。

## 文件结构

```
├── SKILL.md              # 技能定义：触发条件、执行流程、规则索引
├── core/rules.md         # 核心可执行规则库
├── docs/yuanhai_ziping.md # 《渊海子平》完整蒸馏笔记
├── scripts/
│   └── ziping_engine.py  # 排盘计算引擎
├── templates/
│   └── report.md         # 分析报告模板
└── .gitignore
```

## 快速开始

```bash
python3 scripts/ziping_engine.py
```

## 功能

- 四柱八字排盘（年柱、月柱、日柱、时柱）
- 十神配置计算（天干 + 地支藏干）
- 日主强弱分析（得令、得地、得势）
- 格局判断（月令取格）
- 用神忌神确定
- 8步大运排盘
- 流年事件预测（地支冲、伤官见官等）
- 事业/财富/健康分析

## 声明

八字命理不是科学，本技能仅供文化研究参考，不可当真。
