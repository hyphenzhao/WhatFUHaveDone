# 子平正解·书籍蒸馏技能（ziping-zhengliu-skill）

> 基于《渊海子平》蒸馏笔记的八字命理分析技能

## 概述

本技能将《渊海子平》的知识通过"仓颉五维度蒸馏法"（框架、原则、案例、反例、术语）提取为可执行规则，提供完整的八字排盘、格局判断、大运流年分析功能。

## 触发条件

当用户提出以下请求时触发：
- 提供出生日期和时间要求算命/排八字
- 问"我的八字是什么""帮我排盘""看看我的命理"
- 提供年月日时要求做命理分析
- 提到"子平""八字""四柱""算命"

## 执行流程

### Phase 1: 信息采集

向用户确认以下信息：
1. **出生年月日**（公历，注明）
2. **出生时辰**（几点几分，或大概时段）
3. **性别**（用于排大运方向）
4. **出生地**（可选，用于真太阳时校正）

如果用户已提供足够信息，直接进入Phase 2。

### Phase 2: 排盘计算

运行 `scripts/ziping_engine.py` 或直接在对话中执行以下计算：

1. **排四柱**：年柱、月柱、日柱、时柱
2. **定日主**：日干为命主自身
3. **十神配置**：计算各天干地支藏干对应的十神
4. **日主强弱**：得令、得地、得势分析
5. **格局判断**：月令取格
6. **用神忌神**：根据身强身弱确定

### Phase 3: 基于蒸馏笔记的规则匹配

按照以下维度对照蒸馏笔记进行匹配：

#### 框架维度
- 确认格局类型（正格/专旺/特殊/从格）
- 画出推命流程图

#### 原则维度（规则库匹配）
- 匹配R001~R090条断命规则
- 找出适用于该命局的关键断语
- 格式：触发条件 → 断语结论 → 原理说明

#### 反例维度（禁忌检查）
- 检查格局是否破败
- 检查五行是否太过不及
- 检查特殊禁忌（女命、从格等）

#### 术语维度
- 输出十神术语解释
- 输出格局术语解释

#### 案例维度
- 从蒸馏笔记中匹配相似的命例推理链

### Phase 4: 输出完整分析报告

报告结构：

```
# 八字排盘报告

## 基本信息
- 公历：XXXX年X月X日 X时X分 男/女命
- 四柱：年柱、月柱、日柱、时柱
- 日主：X（五行）

## 十神配置
[表格：各柱天干十神 + 地支藏干十神]

## 核心格局判断
- 格局：XX格
- 日主强弱：偏旺/偏弱/中和
- 用神：XX
- 忌神：XX
- 格局层次：上/中上/中/下

## 事业方向
[基于十神组合的职业建议]

## 财富层次
[基于财星数量+日主承载力+大运的判断]

## 健康重点
[基于五行失衡的疾病推断]

## 大运排盘
[8步大运表格，标注喜忌]

## 大运走势分析
[分阶段：青少年/壮年/中年/晚年的运势总评]

## 流年关键事件
[重点流年（带地支冲、伤官见官等标志的年份）]

## 人生阶段总评
[各阶段大运评价表格]

> 声明：八字命理不是科学，以上分析仅供文化参考，不可当真。
```

## 边界与限制

- 本技能的排盘算法为**简化版**，非天文精确计算
- 节气切换以近似日期为准，非精确时刻
- 真太阳时校正未实现
- **命理不是科学，结果仅供文化参考**
- 不提供具体的人生决策建议（如是否结婚、是否投资等）
- 不对未成年人做命运预测
