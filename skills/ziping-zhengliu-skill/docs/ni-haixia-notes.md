# 倪海厦·子平 / 八字相关观点笔记

> 用途：当用户明确要求“按倪海厦/天纪口径”讨论八字、子平时，先读本笔记。
> 说明：以下内容主要整理自课程笔记转载、纪念站转载与二手摘要，不等同于逐字稿。请把它当作“方法论线索”，不要包装成无争议原典。

## 高置信主题

### 1. 他承认子平四柱是重要底盘，但天纪里并不总按 orthodox 子平一路讲到底
- 二手整理常见说法：排盘仍用子平四柱，但正式推演时，他会明显加入皇极经世、卦气、时间节律等自己的讲法。
- 可用于技能中的稳妥表述：**若用户要求倪海厦风格八字，不要只复制传统子平格局论，也要提示他常把八字放进更大的时空推演框架里。**

### 2. 他不把八字当成纯宿命工具
- 多份整理里都能看到类似意思：八字首先是帮助人认清自身角色、方向与风险，而不是拿来把人生判死。
- 可用于技能中的稳妥表述：**八字的价值在于辨方向、避风险、顺势而为，而不是宣告绝对命定。**

### 3. 他强调“命”不是全部，运、局、环境、人事会改写表现
- 二手材料常见总结：命只占一部分，后天运势、环境布局、职业路径、人的选择都会改变结果呈现。
- 可用于技能中的稳妥表述：**命盘给出底子，现实选择决定落点。**

## 中等置信的二手整理观点

### 4. 他区分子平与紫微的切入点
- 常见课程整理会概括成：子平重四柱与日主系统，紫微重星曜与时空结构；两者可以殊途同归。
- 这类内容适合写成“课程笔记归纳”，不要当成逐字原话。

### 5. 他更看重能否用于现实决策，而非术语是否繁复
- 二手整理常把他的风格总结为：抓关键、重应期、重人生阶段判断，不迷失在术语堆叠中。
- 这适合转译成技能约束：**先讲人能怎么做，再讲命局概念。**

   - 八字不应讲成纯宿命
