# 第三部分：断命规则库

> 规则格式：**触发条件 → 断语结论 → 原理说明 → 边界限制**

## 3.1 总纲规则

### R001 日主法则
- **条件**：分析任何命局
- **结论**：以日干为主，专论财官
- **原理**：日干代表命主自身，是一切推论的出发点；财为官之本，官为身之护
- **边界**：无
- **来源**：《玄机赋》《挈要捷驰玄妙诀》《四言独步》

### R002 月令取格法则
- **条件**：月支藏干透出天干
- **结论**：月支取格，乃分贵贱
- **原理**：月令是命局中最有力的地支，决定格局
- **边界**：有官莫寻格局；有格局喜官星
- **来源**：《玄机赋》

### R003 中和法则（最高法则）
- **条件**：分析命局整体
- **结论**：稟中和，莫令太过不及；富贵贫穷，不出乎八字中和之外
- **原理**：五行平衡则吉，偏颇则凶
- **边界**：专旺格和从格除外
- **来源**：《络绎赋》《碧渊赋》

### R004 五行旺衰法则
- **条件**：日干生于某月
- **结论**：
  - 甲乙木最喜春生，壬癸水偏宜冬旺
  - 丙丁火而夏明，庚辛金而秋锐
  - 戊己土要旺四季之期
- **原理**：五行各有所旺之季
- **边界**：得时俱为旺论，失令便作衰看；但四柱无根得时为旺，日干无气遇劫为强
- **来源**：《络绎赋》

## 3.2 身强身弱规则

### R010 身旺喜忌
- **条件**：日干身旺（得令、得地、得势）
- **结论**：
  - 身旺喜：财星、官星、食伤（泄耗克）
  - 身旺印多，喜行财地（财破印）
  - 身旺有杀行印绶，权断之官
  - 身太旺喜见财官
- **原理**：身旺需泄耗克制以求平衡
- **边界**：身旺无依则为僧道之命
- **来源**：《玄机赋》《碧渊赋》

### R011 身弱喜忌
- **条件**：日干身弱（失令、失地、失势）
- **结论**：
  - 身弱喜：印绶、比劫（生扶）
  - 身弱不喜财官
  - 身弱喜印，杀旺无伤；忌行财地
  - 主弱逢印见财星，寻常之客
- **原理**：身弱需生扶以任财官
- **边界**：身弱忌见财官，但若有印绶通关则可
- **来源**：《玄机赋》《身弱论》

### R012 得令与失令
- **条件**：日干生于当旺/不当旺之月
- **结论**：
  - 得令俱为旺论，失令便作衰看
  - 但四柱无根，得时为旺
  - 日干无气，遇劫为强
- **原理**：月令力量最大，但不是唯一决定因素
- **边界**：需综合地支和天干判断
- **来源**：《玄机赋》

## 3.3 十神配合规则

### R020 逢官看财
- **条件**：命局有正官
- **结论**：见财而富贵
- **原理**：财生官，官得财助则有力
- **边界**：官星怕伤官、怕刑冲破害
- **来源**：《玄机赋》《碧渊赋》《挈要捷驰玄妙诀》

### R021 逢杀看印
- **条件**：命局有七杀
- **结论**：遇印以荣华
- **原理**：杀生印，印生身，化杀为权
- **边界**：杀重身轻时最需印绶；杀浅身旺则杀运无妨
- **来源**：《玄机赋》《碧渊赋》《挈要捷驰玄妙诀》

### R022 逢印看官
- **条件**：命局有印绶
- **结论**：遇官十有八贵
- **原理**：官生印，印生身，官印相生
- **边界**：印绶被财破则无功
- **来源**：《玄机赋》《碧渊赋》《挈要捷驰玄妙诀》

### R023 逢财忌杀
- **条件**：命局有财星
- **结论**：有杀十有九贫
- **原理**：财生杀，杀攻身
- **边界**：身旺能任杀者除外
- **来源**：《玄机赋》《碧渊赋》《挈要捷驰玄妙诀》

### R024 伤官用印
- **条件**：伤官格
- **结论**：宜去财
- **原理**：印能制伤，但财破印
- **边界**：身旺者用财，身弱者用印
- **来源**：《玄机赋》

### R025 伤官用财
- **条件**：伤官格
- **结论**：宜去印
- **原理**：财泄伤官生官，但印克财
- **边界**：身旺者用财
- **来源**：《玄机赋》

### R026 伤官伤尽
- **条件**：伤官格，命局无正官
- **结论**：行官运无妨
- **原理**：局中无官，伤官无处可伤
- **边界**：若命局暗藏官星则仍为忌
- **来源**：《玄机赋》

## 3.4 格局成败规则

### R030 正官格
- **成格条件**：官星清纯，有财生、有印护

- **关键断语**：

  - 官贵太盛，旺处必倾

### R031 七杀格
- **成格条件**：有食神制、有印绶化
- **破格条件**：无制化、杀重身轻、制伏太过
- **关键断语**：

  - 杀重身轻，制乡为福
  - 身强杀浅，杀运无妨
  - 杀不离印，印不离杀，杀印相生，功名显达

### R032 财格
- **成格条件**：身旺能任财
- **破格条件**：比劫夺财、财生杀攻身、财星入墓（少许刑冲必发）
- **关键断语**：

  - 财轻勿经劫地，频见妻灾

  - 财星入墓，少许刑冲必发

### R033 印绶格
- **成格条件**：印旺身强
- **破格条件**：财星破印
- **关键断语**：
  - 用之为印不可破
  - 印多者行财而发
  - 印绶被伤，失宗业拋离故里
  - 先财后印，反成其福；先印后财，反成其辱
  - 印绶比肩喜行财乡，印无比肩忌见财伤

### R034 食神格
- **成格条件**：食神有气
- **破格条件**：枭神夺食（偏印克食神）
- **关键断语**：
  - 用之食神不可破（枭印夺食）
  - 食神有气胜财官
  - 食神叠见须忌官乡
  - 七杀喜见食神刑

### R035 伤官格
- **成格条件**：伤官伤尽，或佩印，或生财
- **破格条件**：伤官见官
- **关键断语**：
  - 伤官最怕为官运
  - 伤官见官，格中反忌；不损用神何愁官至
  - 伤官之格，女人最忌；带印带财，反为富贵

# 第六部分：典型案例推理链

> 《渊海子平》以赋文歌诀为主，命例较少。以下从断语反推典型命例逻辑。

## C001 杀印相生格

**假设命局**：日干弱，月令七杀，有印绶透出
**推理链**：
1. 日干弱 → 需要生扶
2. 月令七杀 → 杀克身，更弱
3. 有印绶 → 杀生印，印生身
4. 化杀为权 → 转危为安
**断语**：逢杀看印，遇印以荣华；杀印相生，功名显达
**结果**：贵

## C002 伤官佩印格

**假设命局**：伤官格，有印绶制伤
**推理链**：
1. 伤官格 → 才华外露但克官
2. 有印绶 → 印制伤官
3. 伤官被制 → 才华收敛为学识
4. 印护官 → 官星不被伤
**断语**：伤官用印宜去财；伤官伤尽妙入印财之地
**结果**：贵（学识渊博）

## C003 财多身弱格

**假设命局**：财星极旺，日干虚弱
**推理链**：
1. 财星旺 → 看起来好
2. 日干弱 → 担不起财
3. 财生杀 → 杀攻身
4. 身弱不胜财 → 因财惹祸
**断语**：财多身弱行财运，此处方知下九台；财生杀党夭折童年
**结果**：贫/凶

## C004 从财格

**假设命局**：日干无根，财星遍布
**推理链**：
1. 日干极弱无根 → 无法自立
2. 财星极旺 → 财势主导
3. 弃命从财 → 顺从财势
4. 从财纯粹 → 大富
**断语**：弃命从财须要会财；从财忌杀
**结果**：富（前提是纯粹无破）

## C005 曲直格（木专旺）

**假设命局**：亥卯未三合木局，日干甲乙
**推理链**：
1. 亥卯未全 → 木气专旺
2. 日干甲乙 → 木日元
3. 一气专旺 → 专旺格
4. 木旺宜火 → 喜食伤泄秀
**断语**：亥卯未逢于甲乙，富贵无疑；木旺宜火之光辉
**结果**：贵
